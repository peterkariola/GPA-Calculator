Guiding on Using the GPA Calculator

1. Enter Your Course Code in a format like this PSY202, MTH204

2. Type your grade e.g. A, B, C, D , Just a single character is needed here

3. Type your course unit e.g. 3, 4, 5, 6, 10

4. Please cross check all these values to make sure that they are correct as imputing wrong values will lead to inaccurate calculation of your GPA.

5. If the fields are more than what you needed, just ignore other fields and fill the ones you need, the reason being that number of courses taken by each student are not the same.

6. Don't add numbers where you ought to put alphabets (vice versa)

7. To calculate again for someone else or you want to re-do the calculation again, kindly input your new values and hit Calculate
